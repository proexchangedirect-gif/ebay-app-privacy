Privacy Policy

Last Updated: October 23, 2025

1. Introduction

This application ("eBay Listing Automation") is a personal-use tool designed to automate the creation of eBay listings. This privacy policy describes how we handle data in connection with this application.

2. Application Purpose

This application is used solely by the account owner for personal eBay seller automation. It is not a public application and does not serve other users.

3. Data Collection and Use

The application accesses the following eBay data:

  • Your eBay seller account information
  • Product listing data (titles, descriptions, images, pricing)
  • Inventory information
  • Item specifics and categories

This data is used exclusively to:

  • Create and manage eBay listings on your behalf
  • Automate listing workflows
  • Improve listing quality and efficiency

4. Data Storage

All data is processed locally or temporarily stored only for the duration necessary to complete automated tasks. No personal data is permanently stored, shared with third parties, or used for any purpose other than the stated automation functions.

5. Data Sharing

We do not share, sell, or distribute any data collected through this application. Data is only transmitted between:

  • The application and eBay's API servers
  • Your local automation workflow

6. Account Deletion

If you delete your eBay account, all associated data within this application will be automatically removed. You may also revoke this application's access at any time through your eBay account settings.

7. Security

We implement industry-standard security measures to protect your data. API credentials are stored securely and are never shared or exposed.

8. Third-Party Services

This application integrates with:

  • eBay API (governed by eBay's privacy policy)
  • Google Drive (for image storage, governed by Google's privacy policy)

9. Your Rights

You have the right to:

  • Access any data processed by this application
  • Request deletion of your data
  • Revoke application access at any time
  • Modify or update your preferences

10. Contact

For questions about this privacy policy or data handling practices, you may contact the application owner through eBay's messaging system or at proexchangeDirect.com.

11. Changes to This Policy

This privacy policy may be updated periodically. Any changes will be reflected with an updated "Last Updated" date.

12. Compliance

This application complies with:

  • eBay Developer Program policies
  • eBay's Marketplace Account Deletion requirements
  • Applicable data protection regulations
